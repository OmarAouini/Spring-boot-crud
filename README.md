# Spring-boot-crud
demo api with crud operations and Authentication, and multiLanguage error messages.<br>
example Unit/integration test are included.<br>

technologies used:

-Spring boot<br>
-Apache maven<br>
-MongoDB<br>
-Junit5<br>
-Mockito<br>
-Google Truth (test assertions Library) <br>
-Apache Poi (Excel/csv creation) <br>
-Mapstruct<br>
-Swagger2<br>
-Docker<br>
-Jacoco (code coverage)<br>
